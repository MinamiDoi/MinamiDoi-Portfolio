Cafe Azito サイトリニューアル（架空）

ターゲット

20代〜30代の非日常的な空間でリラックスしたい方やおしゃれな雰囲気を楽しみたい方

目的

来店数を増やすため。新規・リピーター様の獲得

情報設計

⽬の前の海を全⾝に感じられる立地とリラックスできるテラスや店内の雰囲気を前面にアピールすること。 カフェだけではなくサロンやスタジオも併設しているので、⽇常とは少し違った特別な時間と空間を過ごせることが伝わるサイト構成を心がけました。

デザイン

「スロウでメロウな⼤⼈の空間つくり」がお店のコンセプトになっているので、柔らかい「白」と明るい「水色」で開放感と優しさで柔和な印象付けを図りました。 さらに余白を多く取り開放感の演出と、文章よりも写真を多めに掲載することでサービスを想像しながら見入ってしまうようなレイアウトを意識しました。

フォント : Quattrocento / Adobe Handwriting Ernie / cursive / Noto Sans / Noto Sans JP

制作範囲・期間

企画・ワイヤーフレーム：１週間

デザインカンプ：１週間

コーディング：１日半

使用ツール

Visual Studio Code / XD / Photoshop
